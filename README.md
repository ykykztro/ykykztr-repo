# WindowQuickJump

窗口/标签页编号直达工具 — 用快捷键将窗口绑定到数字槽位，一键跳回。

## 功能

- **9 个编号槽位 (Slot 1~9)**：把任意窗口钉到数字上，随时切换
- **浏览器标签页定位**：跳转到浏览器时自动轮询标签页，找到绑定时所在的那一页
- **屏幕中央提示**：所有操作反馈以 ToolTip 显示在屏幕中央，自动消失
- **自动管理员提权**：部分窗口需管理员权限才能激活，脚本启动时自动请求
- **失效检测**：绑定的窗口若已关闭，自动清空槽位并提示

## 快捷键

| 快捷键 | 作用 |
|--------|------|
| `Alt + 1~9` | **智能切换**：槽位为空 → 绑定当前窗口；槽位已有窗口 → 跳转到它 |
| `Ctrl + Alt + 1~9` | **强制覆盖**：用当前窗口覆盖绑定，无论槽位是否已有内容 |

## 使用流程

1. 切到想记住的窗口（如 VS Code），按 `Alt+1` → 提示「Slot 1 已绑定」
2. 切到浏览器某标签页，按 `Alt+2` → 绑定
3. 在任意窗口按 `Alt+1` → 立刻跳回 VS Code；按 `Alt+2` → 跳回浏览器并自动定位到当初的标签页
4.想替换槽位 1 的绑定，按 `Ctrl+Alt+1` 强制覆盖

## 浏览器标签页说明

绑定浏览器时记录的是**窗口标题（即当前标签页标题）**。跳转时：

- 当前标签页标题已匹配 → 不切换，直接提示
- 否则用 `Ctrl+Tab` 最多轮询 30 次找到标题匹配的标签页
- 支持的浏览器类名：`Chrome_WidgetWin_1` / `Chrome_WidgetWin_0`（Chrome、Edge 等 Chromium 内核浏览器）

> ⚠️ 同一浏览器窗口中如果有两个标题完全相同的标签页，无法精确区分。

## 限制

- 不允许绑定 AHK 自身窗口（会提示「不能绑定此窗口」）
- 脚本退出时自动清空所有槽位

## 安装与启动

1. 安装 [AutoHotkey v2](https://www.autohotkey.com/)
2. 双击 `WindowQuickJump.ahk` 运行
3. 屏幕中央提示启动信息，之后常驻后台
